# 🏪 Hardware Sales Dashboard | Power BI

An interactive sales analytics dashboard built in **Power BI** to help analyze the sales performance of a hardware company. The project uses a MySQL database created from an SQL dump file, followed by data cleaning in Power Query and KPI creation using DAX.

## 📌 Project Overview

This project is based on the **AtliQ Hardware Sales Insights** case study. The company was facing challenges in monitoring sales performance because management relied on verbal updates and large Excel reports, making it difficult to identify declining markets, customers, and revenue trends. 

The objective of this project is to transform raw sales data into an interactive dashboard that provides clear business insights for better decision-making.

## 📊 Dashboard Features

* Total Revenue KPI
* Total Sales Quantity KPI
* Revenue Trend Analysis
* Revenue by Market
* Sales Quantity by Market
* Top 5 Customers by Revenue
* Product Type Revenue Distribution
* Interactive Year and Month Filters

---

## 🔄 Project Workflow

1. Imported the provided SQL dump into **MySQL**.
2. Connected **Power BI** to the MySQL database.
3. Cleaned and transformed the data using **Power Query**.
4. Created DAX measures:

   * Revenue
   * Sales Quantity
5. Designed an interactive dashboard to visualize sales performance.
